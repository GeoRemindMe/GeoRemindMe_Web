GeoAuth Backend¶

class geoauth.backend.OAUTH_Store¶

Handle the models in the datastore (clients, tokens, etc)

authorize_token(token, user)¶

Authorizes a request token, sets the verifier and the user in the token

Parámetros:
  • token (oauth2.Token) – Request token to be authorized
  • user (georemindme.models.User) – User who authorizes the token
Devuelve:

oauth2.Token authorized
Raises :

OAUTHException if the token key is invalid
fetch_appInfo(key)¶

Returns a dict with the name and description of a client application

Parámetros:key (string) – App key in the datastore
Devuelve:dict with the info of the app
Raises :OAUTHException if token key is invalid
fetch_consumer(oauth_request)¶

Load the consumer (key, secret) who is doing a request

Parámetros:oauth_request (oauth2.Request) – oauth request
Retuns :oauth2.Consumer
Raises :OAUTHException if the client key is invalid
fetch_token(oauth_request)¶

Get the token object in a request

Parámetros:oauth_request (oauth2.Request) – oauth request
Devuelve:oauth2.Token
Raises :OAUTHException if token is invalid
fetch_token_db(oauth_request)¶

Get the token object in a request

Parámetros:oauth_request (oauth2.Request) – oauth request
Devuelve:oauth2.Token and geoauth.models.OAUTH_Access
Raises :OAUTHException if token is invalid
generate_access_token(token, oauth_consumer)¶

Generate the new access token

Parámetros:
  • token (oauth2.Token) – Request token authorized
  • oauth_consumer (oauth2.Consumer) – Application requesting the access token
  • user (georemindme.models.User) – User for the authorized token
Devuelve:

oauth2.Token
Raises :

OAUTHException if the token key or consumer key are invalid
generate_request_token(oauth_request, oauth_consumer, oauth_callback=None)¶

Load the request token in the request

Parámetros:
  • oauth_request (oauth2.Request) – oauth request
  • oauth_consumer (oauth2.Consumer) – oauth consumer
  • oauth_callback (string) – callback URL
Devuelve:

oauth2.Token with the new key and secret
Raises :

OAUTHException if client key is invalid
