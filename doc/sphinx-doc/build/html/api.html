georoute.api.MIN_PATH_LENGTH¶

Minimun number of points needed to add or search a path

value: 5

georoute.api.add_path(user, points)¶

Add a new path for the user The order of the points is important, two paths with the same points but with different order, are different paths

Parámetros:
  • user (georemindme.models.User) – The user adding the path
  • points (db.GeoPt) – List of points, min. length: MIN_PATH_LENGTH
Devuelve:

RouteUserPath – The RouteUserPath added
Raises :

NoPoints, AttributeError
georoute.api.add_point(user, point)¶

Add a new RouteUserPoint for the user. If the user already has this point, returns it

Parámetros:
  • user (georemindme.models.User) – The user adding the point
  • point (db.GeoPt) – The point you want to add
Devuelve:

RouteUserPoint – The RouteUserPoint added
Raises :

AttributeError
georoute.api.get_path(path)¶

Get the path

Parámetros:path (db.Key or string) – The path’s key
Devuelve:georoute.models.RoutePath or None
Raises :KindError
georoute.api.search_path(user=None, points=None, sort=True, aprox=True)¶

Search all the possible paths for the user and points Search paths from this user and all the others system’s paths If user is None, search all the paths for those points

Parámetros:
  • user (georemindme.models.User) – The user looking for his path
  • points (db.GeoPt) – List of points, min. length: MIN_PATH_LENGTH
  • sort (boolean) – If you want to get the paths sorted in a list or a dictionary
  • aprox (boolean) – If you want some points to be added
Devuelve:

list of tuples (path_key, occurrences) ordered by the most probable paths
Devuelve:

dictionary {path_key: occurrences}
Raises :

NoPoints, AttributeError
